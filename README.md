# UmeYui — 古民家梅屋 予約・交流プラットフォーム

古民家梅屋のチャレンジショップ向けの招待制コミュニティアプリ。  
「一人じゃないから、はじめられる」をコンセプトに、出店者同士が集まって開催日を確定させる条件付き予約システムを実装しています。

---

## 機能概要

### 1. 最低開催枠数システム

「〇人集まれば出店する」という条件付き予約。目標人数に達した時点で開催確定となり、参加者全員にプッシュ通知が届きます。発起人は募集中に最小・最大人数を変更でき、最小人数を下げた時点で条件を満たせば自動確定します。

### 2. 参加申請フロー

発起人が枠を作成し、他の出店者が参加申請を送る申請制。発起人が承認・却下を管理します。

### 3. 相乗りチャット

開催確定した枠ごとに専用チャットルームを自動生成。確定参加者のみがアクセスできます。

### 4. プッシュ通知

Firebase Cloud Messaging（FCM）によるプッシュ通知。開催確定・参加申請・メッセージ受信などのタイミングで通知が届きます。

### 5. パスワードリセット

メールアドレスに確認コード（6桁・10分有効）を送信する2ステップのリセットフロー。リセット後は既存セッションをすべて無効化します。

### 6. ホームページ連携

umeya.life（Astro + Vercel）向けの認証不要パブリックAPI。開催確定イベント一覧・出店者一覧を提供します。プロフィール更新やイベント確定時にVercel Build Hookを呼び出してホームページを自動再ビルドします。

### 7. 強制アップデート

App Storeの公開バージョンと比較し、古いバージョンのアプリはアップデートを強制するダイアログを表示します。

### 8. 管理者機能

招待制アカウント発行・有効/無効切り替え・パスワードリセット・メールアドレス変更（復旧用）。

## 技術スタック

| レイヤー            | 技術                                        |
| ------------------- | ------------------------------------------- |
| モバイルアプリ      | Flutter（iOS / Android）                    |
| バックエンドAPI     | Cloudflare Workers + Hono                   |
| データベース        | Cloudflare D1（SQLite互換）                 |
| ストレージ          | Cloudflare R2（アバター・ホームページ画像） |
| メール送信          | Resend                                      |
| プッシュ通知        | Firebase Cloud Messaging（FCM）             |
| ホームページ        | Astro + Vercel（umeya.life）                |
| 認証                | JWT（PBKDF2ハッシュ + token_version無効化） |

## セキュリティ

- パスワード: PBKDF2（100,000イテレーション）でハッシュ化
- JWT: token_version によるログアウト・パスワードリセット時の即時無効化
- レート制限: ログイン試行・パスワードリセット要求をIP別に制限
- 権限制御: admin / vendor ロールによるエンドポイントアクセス制御
- SQLインジェクション: prepared statement で対策済み
- パブリックAPI: CORS を umeya.life のみに限定
